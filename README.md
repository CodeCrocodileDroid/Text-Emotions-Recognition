# Text-Emotions-Recognition
Python Machine Learning

Key Features:
Data Loading & Exploration: Loads the CSV file and shows emotion distribution

Text Preprocessing:

Lowercasing

URL removal

Punctuation removal

Stopword removal

Lemmatization

Tokenization

Feature Extraction:

TF-IDF vectorization

Count vectorization

N-gram support (unigrams + bigrams)

Multiple ML Models:

Naive Bayes

Logistic Regression

Support Vector Machines (SVM)

Random Forest

K-Nearest Neighbors

XGBoost

Neural Network (MLP)

Evaluation & Visualization:

Accuracy scores

Classification reports

Confusion matrices

Model comparison plots

Prediction Functionality:

Predict emotions for new text

Show prediction probabilities

Feature Analysis:

Identify important words for each emotion

Usage Instructions:
Save the dataset as emotion_dataset_raw.csv in the same directory
